# 星迹 StarryChat

一个用 Supabase 做的在线聊天室。【11岁开始写的？！？】

## 能干嘛

- 注册登录，记住我
- 建房间、进房间
- 发文字、图片、文件
- @ai 问问题
- 私聊、视频通话
- 搜索消息
- 深色/浅色主题

## 怎么跑

把 `index.html` 扔到 GitHub Pages 就行。Supabase 配置在代码里。

## 管理后台

访问 `admin.html`，密码自己看代码。

## 为什么做这个

练手。学着用 Supabase，学着处理实时数据。
